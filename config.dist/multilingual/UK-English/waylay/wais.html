WAIS is not commonly supported by World Wide Web browsers, and so you may find that you have difficulty in using this URL.

Typically, browsers may be configured to speak the WAIS protocol via a proxy server, such as the CERN httpd. There are also some WWW based gateways to WAIS. Check out the WAIS, Inc WWW server for more information about these.
