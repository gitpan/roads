Editor

This is the start of a series of FORMs that allows you to generate an IAFA template from the comfort of your WWW browser. This form lets you either specify the type of the template that you wish to create or edit an existing template. In the former case the system loads in a blank outline template for you to start editing. In the latter case you must specify the handle of the template to edit (you may find it easier to use the admin.pl program for locating templates for editing).
